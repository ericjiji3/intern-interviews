My Video Game List
This was my final project for my Web Programming course, where me and two other teammates made a fully functional website. The website's name is My Video Game List, which is a database that keeps reviews and ratings on games from several different consoles. Not only does it serve as a database for previous reviews and ratings, but it also allows users to create an account and leave their own reviews and ratings. It also has an interactive tier list maker that was primarily created from JavaScript. My role during this project was primarily developing HTML, JavaScript, and CSS functionality for our home, console, and log in/sign up pages. I spent the most time developing our console page to load images in JavaScript from metacritic, using python to format the image URLs into figure tags which are stored in a text file.
Space Raiders
This was a game I developed with two other teammates called Space Raiders, which plays similar to Space Invaders. It is a browser game that uses Phaser3, a HTML5 game framework. Most of the files for this game are JavaScript files, but use the syntax from the Phaser3 framework. I took on multiple roles for this project, including developing the mechanics and creating character models and maps with assets found online. The game includes interchangeable scenes, which are different levels in the game, and mechanics like power ups, player and enemy movement, and shooting.
